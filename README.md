# gregMod.HexViewer - Hardware Inspector

> Inspect raw hexadecimal data of in-game hardware components.

**Author:** MLeeM97 (teamGreg) | **License:** MIT | **Framework:** [gregCore](https://git.datacentermods.com/teamGreg/gregCore)

---

## Features
- **F1** to toggle Hexadecimal Viewer overlay
- JADE-inspired dark UI design
- InputSystem compatible (no legacy Input)

## Installation

1. Install **MelonLoader** (v0.6+)
2. Place `gregCore.dll` + `gregMod.HexViewer.dll` into `Game/Mods/`
3. Start the game and press **F1**

## Dependencies

- `gregCore.dll`

## Building from Source

```bash
cd gregFramework/src\gregMod.HexViewer
dotnet build -c Release
# Output: bin/Release/net6.0/gregMod.HexViewer.dll
```

Or build everything at once:

```bash
cd gregFramework/deploy
./build-all.ps1
# Output: gregFramework/BuiltModsForGame/
```
